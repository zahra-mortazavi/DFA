# 📘 README – Deterministic-Finite-Automaton (Java)

## 📌 Project Introduction

This project is a console-based program written in **Java** that checks whether an input string can be generated by a **Regular Grammar**, either **Left Linear** or **Right Linear**.

The program first receives the grammar from the user, validates it, and then checks multiple input strings. The result is displayed as `YES` or `NO`.

---

## ⚙️ How the Program Works

### 1️⃣ Selecting the Grammar Type

At the beginning, the user is asked:

```
1) left liner
2) right liner
```

* If option `1` is selected → The grammar is **Left Linear**
* If option `2` is selected → The grammar is **Right Linear**

If a Left Linear grammar is selected, the input string is reversed before being processed.

---

### 2️⃣ Entering Grammar Rules

The user must enter the number of grammar lines and then provide each rule in the following format:

```
S=aB|c
```

General format:

```
NonTerminal = production1 | production2 | ...
```

### Validation Rules:

* The left-hand side must be a single uppercase English letter (`A-Z`).

* Terminals must be lowercase English letters (`a-z`).

* In a **Right Linear Grammar**:

  ```
  A → aB
  A → a
  ```

* In a **Left Linear Grammar**:

  ```
  A → Ba
  A → a
  ```

If the grammar is invalid, the program prints:

```
grammar is wrong
```

and terminates.

---

### 3️⃣ Checking Strings

After defining the grammar, the user can enter multiple strings.

* If the string can be generated → `YES`
* If it cannot → `NO`
* If `$` is entered → It is treated as the empty string (epsilon).
* If `END` is entered → The program terminates.

---

## 🧠 Implementation Logic

### ✅ Data Structures

* `Map<Character, List<String>>` is used to store grammar rules.
* `Stack<Character>` is used to process and validate the input string.
* The start symbol is `S`.

---

### ✅ String Validation Algorithm (`checkString`)

1. Push `S` onto the stack.
2. Compare input characters with the top of the stack.
3. Apply matching grammar rules.
4. If the stack is empty at the end → The string is valid.
